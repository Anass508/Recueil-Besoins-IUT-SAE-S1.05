#  SAÉ S1.05 : Recueil de besoins - Réussite en IUT

Ce projet vise à comprendre et formaliser les attentes des étudiants de l'IUT de Villetaneuse pour identifier les paramètres facilitant la réussite de leurs études.

##  Objectifs
1. **Phase exploratoire** : Formuler des hypothèses sur les facteurs de réussite.
2. **Entretiens** : Réalisation d'entretiens (minimum 2 par étudiante) pour recueillir des témoignages directs.
3. **Analyse** : Synthèse des besoins via des tableaux, graphiques et la création de **Personas**.
